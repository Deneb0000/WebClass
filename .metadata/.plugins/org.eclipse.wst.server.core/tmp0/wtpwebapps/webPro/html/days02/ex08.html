1. abbr
	- abbreviation 약어
	- 마우스를 올리면 풍선도움말(툴팁), 밑줄(점선)
	- 브라우저 검색엔진 제공
2. address
	- 문저나 기사의 작성자/소유자의 연락처 정보(주소) 정의하는 태그
	- 연락처 정보 : 이메일, 전화번호, 실제 주소, url, 소셜 미디어.. 등등
	- 자동적으로 기울림꼴로 표시가 된다.
3. cite
	- 창작작품 ( 책, 시, 노래, 영화, 그림 등등 -작가명은 포함되지 않는다)의 제목을 정의한다
	- 일반적으로 기울림꼴로 표시가 된다.
4. bdo
	- [b]i-[d]irectional [O]verride
		양 방향				재정의
	- 현재 텍스트 방향을 재정의하는 태그
	- ltr : 왼쪽에서 오른쪽으로   // - rtl : 오른쪽에서 왼쪽으로
