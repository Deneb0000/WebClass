//배열에서 사용되는 함수들 Array length : 배열의 크기 (속성) Array toString() : 배열의 요소를 콤마 구분자로 문자열 반환 Array at() : Array join() Array push() : 배열의 새로운 요소를 마지막에 추가하고 배열크기를 반환하는 메서드 Array pop() : 배열의 마지막 요소를 제거하고 반환하는 메서드 Array shift() : 첫번째 요소를 제거하고 반환하는 메소드 + 모두 왼쪽으로 이동 Array unshift() : 첫번째 요소에 삽입을 하고 모든 요소를 오른쪽으로 이동하고 배열 크기를 반환하는 메서드 Array delete() : 요소 삭제 Array concat() : 배열을 병합할때 사용한다 Array copyWithin() : 배열의 요소를 배열의 다른 위치에 복사하는 메서드 (덮어쓰기) 배열크기는 바뀌지 않는다 Array flat() : 평면화 ***, 배열의 차원을 줄이는 메서드. 다차원배열 -> 1차열 변환 Array splice() Array toSpliced() : 원래 배열은 변경하지 않고 새로운 배열을 반환하는 메서드 Array slice()
